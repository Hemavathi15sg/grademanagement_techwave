# Techwave Go Backend Starter

A minimal, extensible Go backend for experimenting with GitHub Copilot and DevOps workflows.

## Features

- Student and Grade domain with CRUD operations
- RESTful API using Gorilla Mux
- Repository and handler patterns
- Unit test samples
- Ready for Copilot-driven development and demo scenarios

## Prerequisites

- Go 1.20+ installed

## Run the Server

```sh
go run main.go
